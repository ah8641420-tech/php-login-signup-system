# PHP Login & Registration System

A secure, complete authentication system built with PHP and MySQL featuring user registration, login, password reset, and session management.

## ✨ Features

- 🔐 **User Registration** - Secure account creation with email validation
- 🔑 **User Login** - Session-based authentication
- 🔒 **Password Hashing** - Using PHP's `password_hash()` function
- 🔄 **Password Reset** - Token-based password recovery system
- 📱 **Responsive UI** - Built with Tailwind CSS
- 🛡️ **Security** - SQL injection prevention, XSS protection
- 👤 **User Dashboard** - Protected member area

## 🛠️ Technologies Used

- **Backend**: PHP 7.4+
- **Database**: MySQL
- **Frontend**: HTML5, Tailwind CSS
- **Server**: XAMPP/WAMP/LAMP

## 📋 Prerequisites

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web server (Apache recommended)
- XAMPP/WAMP/LAMP for local development
